Testing & Deployment

Building & Testing

Build the project

npm run build

This will create the build in the dist folder.

To test locally

npm run start

To test on network

firebase serve

Deploying in Live Server

Run the following commands:

npm install -g firebase-tools
firebase init
fireabase deploy
